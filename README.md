# CAREOS - The AI Layer for Healthcare

Premium frontend-only healthcare SaaS website built with React, Vite, Tailwind CSS, React Router, Framer Motion, Lucide React, and Recharts.

## Run

```bash
pnpm install
pnpm run dev
```

## Routes

- `/` - Public CAREOS landing page
- `/city/:citySlug` - City healthcare intelligence page
- `/hospital/:hospitalSlug` - Hospital profile and doctors page
- `/admin/login` - Admin login
- `/admin/dashboard` - Admin dashboard
